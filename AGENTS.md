# Agent Instructions

- Next.js 개발 서버(dev server)를 열 때는 항상 포트 `3000` 번부터 사용합니다. (`next dev`)
- 해야할 일과 작업 현황은 항상 `todo.md` 파일을 통해 기록하고 관리해야 합니다.
- 이전 세션의 내용과 컨텍스트는 `memory.md` 파일에 기록해두어 새로운 세션에 이어갈 수 있게 해야 합니다. 새로운 작업을 시작할 때 이 파일을 참조하세요.
- 코드/설정 수정 후에는 반드시 `agent-browser`로 브라우저 검증을 수행하고, 실패 시 원인 확인 후 재시도한다.
- 사용자가 UI/라우팅/로그인 문제를 제기한 경우, `agent-browser` 결과(접속 URL, 에러 로그, 스냅샷)를 확인한 뒤에만 답변한다.
