# 🎤 Neon Shaman Pop

## AI Virtual Idol Platform -- Brand & UI Design System (v1.0)

------------------------------------------------------------------------

# 1. 브랜드 철학

## 🎯 Core Statement

> "무대 위 네온 오라를 두른 가상의 아이돌을 관리하는 디지털 샤먼
> 스테이지."

### 핵심 방향

-   UI는 모던·미니멀
-   무드는 네온 샤먼(K-판타지)
-   감정은 무대의 긴장감 + 신비로운 에너지
-   디자인은 어둡고 차분하지만, 포인트는 선명

------------------------------------------------------------------------

# 2. 시각적 DNA (Visual DNA)

## 2.1 Color System (Dark Mode Only)

### 🎨 Base Layer (90%)

-   Background Primary: `#0B0F14`
-   Background Secondary: `#121722`
-   Surface Card: `#161C26`
-   Divider: `rgba(255,255,255,0.06)`

기본은 깊은 블루-블랙 톤 유지

------------------------------------------------------------------------

### ⚡ Accent Layer (8%)

네온은 항상 선(line) 또는 외곽선으로만 사용

-   Primary Neon Pink: `#FF2F92`
-   Neon Crimson: `#FF3B4A`
-   Neon Cyan: `#00F0FF`
-   Neon Violet: `#8A5CFF`

------------------------------------------------------------------------

### ✨ Ritual Highlight (2%)

팬레벨 및 랭킹 전용

-   Hologram Gradient: `#FF2F92 → #8A5CFF → #00F0FF`
-   Glow Shadow: `0 0 12px rgba(255, 47, 146, 0.6)`

------------------------------------------------------------------------

# 3. Typography System

## Primary Font

-   Pretendard / Inter / SF Pro
-   Weight: 400--600
-   700 이상 사용 금지

## Text Colors

-   Primary: `#FFFFFF`
-   Secondary: `rgba(255,255,255,0.65)`
-   Caption: `rgba(255,255,255,0.45)`

네온 텍스트 사용 금지 (아이콘/라인에만 사용)

------------------------------------------------------------------------

# 4. UI Mood Principles

## 공간감

-   여백 충분히 확보
-   카드 radius 16px
-   과도한 그림자 금지
-   Glow는 매우 제한적으로 사용

## 네온 사용 규칙

❌ 버튼 전체 네온 채움 금지\
❌ 배경 네온 패턴 금지\
❌ 과도한 그라데이션 금지

✅ 활성 상태 표시\
✅ 팬레벨 배지\
✅ 랭킹 상위 하이라이트

------------------------------------------------------------------------

# 5. Symbol System

## 부적 스티커 (Core Element)

형태: - 세로 직사각형 - 상단 매듭 디테일 - 내부 추상 문양
(불꽃/별/소용돌이)

사용 위치: - 팬레벨 배지 - 랭킹 트로피 옆 - 캐릭터 프로필 상단

컬러: Neon Pink + Glow

------------------------------------------------------------------------

# 6. Screen Guidelines

## 6.1 메인 피드

-   풀스크린 카드 구조
-   우측 인터랙션 버튼
-   하단 캐릭터명 + 캡션

Neon 적용: - 좋아요 활성 시 네온 - 팔로우 시 이름 하단 네온 라인 -
좌하단 부적 워터마크

------------------------------------------------------------------------

## 6.2 인기 랭킹 페이지

-   상단 리그 표시 네온 라인 강조
-   1\~3위 Glow 배지
-   나머지 심플 카드 유지

------------------------------------------------------------------------

## 6.3 캐릭터 생성 화면

-   선택 카드 활성 시 네온 외곽선
-   생성 버튼 1개만 Solid Neon Pink 허용

------------------------------------------------------------------------

## 6.4 캐릭터 관리 화면

-   팬레벨 옆 부적 아이콘
-   얇은 네온 진행 게이지

------------------------------------------------------------------------

## 6.5 캐릭터 상세 페이지

-   상단 약한 무대 조명 그라데이션
-   팔로우 버튼 네온 Outline
-   팔로우 시 Glow

------------------------------------------------------------------------

# 7. Motion System

-   Fade 200ms
-   Scale 1.02 max
-   팬레벨 업 시 Glow Pulse

------------------------------------------------------------------------
